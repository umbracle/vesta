---
title: Catalog list
---

The `catalog list` command is used to list all the node types available to deploy.

Then, you can use the [`inspect`](/docs/cli/catalog-inspect) command to show the specific information of each plugin.

## Usage

```shell-session
$ vesta catalog list
```

## Examples

```shell-session
$ vesta catalog list
Name
Lighthouse
Prysm
Teku
Besu
Geth
Nethermind
```
